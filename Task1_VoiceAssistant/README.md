# Voice Assistant

## Description
A Python Voice Assistant developed for the Oasis Infobyte Internship.

## Features
- Voice Output
- Greeting
- Open Google
- Open YouTube
- Open GitHub
- Open ChatGPT
- Google Search
- Current Time
- Current Date
- Exit Command

## Technologies Used
- Python
- edge-tts
- Wikipedia
- Requests
- Webbrowser

## Run the Project

pip install -r requirements.txt

python main.py
