# Weather App

## Description
A Weather App developed using Python, Tkinter, and the OpenWeather API for the Oasis Infobyte Internship.

## Features
- Search weather by city
- Temperature
- Feels Like Temperature
- Humidity
- Wind Speed
- Weather Condition
- Modern GUI

## Technologies Used
- Python
- Tkinter
- Requests
- OpenWeather API

## How to Run

```bash
py main.py
