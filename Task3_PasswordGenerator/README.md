# Password Generator

## Description
A Password Generator developed using Python Tkinter for the Oasis Infobyte Internship.

## Features
- Generate strong passwords
- User-defined password length
- Copy password to clipboard
- Modern GUI
- Easy to use

## Technologies Used
- Python
- Tkinter
- Random
- String

## How to Run

```bash
py main.py
```
