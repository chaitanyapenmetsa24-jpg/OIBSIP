# BMI Calculator

## Description
A simple and modern BMI Calculator developed using Python Tkinter for the Oasis Infobyte Internship.

## Features
- Calculate Body Mass Index (BMI)
- Displays BMI Category
- Modern User Interface
- Reset Button
- Error Handling

## Technologies Used
- Python
- Tkinter

## How to Run

```bash
py main.py
```
